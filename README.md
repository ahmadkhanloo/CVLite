# CVLite Resume Builder

رزومه‌ساز محلی برای import فایل‌های Markdown و JSON شبیه RxResume، ویرایش کامل فیلدها، preview و دانلود مستقیم PDF.

## اجرا

```powershell
npm start
```

بعد از اجرا، برنامه روی این آدرس در دسترس است:

```text
http://127.0.0.1:4173
```

اگر پورت `4173` از قبل در حال استفاده باشد، برنامه به‌صورت خودکار پورت‌های بعدی مثل `4174` را امتحان می‌کند و آدرس درست را در ترمینال چاپ می‌کند.

## قابلیت‌ها

- Import فایل‌های `.json`, `.md`, `.markdown`
- ویرایش اطلاعات اصلی، Summary، مهارت‌ها، تجربه، تحصیلات، پروژه‌ها، دوره‌ها، زبان‌ها، علایق، انتشارات، دستاوردها و بخش‌های سفارشی
- افزودن، حذف، مخفی کردن و جابه‌جایی آیتم‌ها
- آپلود عکس پروفایل با crop مربعی مرکزی
- انتخاب زبان رابط کاربری بین فارسی و انگلیسی
- انتخاب تم روشن، تیره یا مطابق سیستم
- ذخیره خودکار draft در `localStorage`
- خروجی گرفتن JSON و Markdown
- دانلود مستقیم PDF با Edge یا Chrome headless

## فونت‌ها

فونت‌های برنامه به‌صورت local داخل پروژه قرار دارند تا UI و PDFها بدون وابستگی به اینترنت رندر شوند:

- `Vazirmatn` برای متن‌های فارسی
- `Inter` برای متن‌های انگلیسی

## قالب‌ها

- `Dark Sidebar`: مشابه رزومه دو ستونه با ستون چپ تیره
- `Classic Blue`: مشابه قالب سفید با خطوط آبی و عکس بالا-راست
- `Purple Compact`: مشابه قالب فشرده با حاشیه بنفش

## PDF Export

سرور مسیر مرورگرهای رایج ویندوز را خودکار پیدا می‌کند. اگر مرورگر در مسیر دیگری نصب شده بود، متغیر محیطی زیر را تنظیم کنید:

```powershell
$env:CVLITE_BROWSER = "C:\Path\To\chrome.exe"
npm start
```
